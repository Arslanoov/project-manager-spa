<p align="center"><b>Start the project</b></p>

<ol>
    <li><a href="https://github.com/Arslanoov/project-manager-api">Run the API</a></li>
    <li>Clone this repo</li>
    <li>Run:</li>
</ol>

    make serve
    make install-deps

<p align="center"><b>Run tests</b></p>

    make test

<p align="center"><b>Build project</b></p>

    make build
